### postMessage实例

使用postMessage演示跨域，首先需要两个不同域，这里server1和server2是本地建立的两个不同域，分别为`http://127.0.0.1:81`和`http://127.0.0.2:8181`

安装好node以后，启动这两个服务`node server1.js`、`node server2.js`

打开页面`http://127.0.0.1:81/a.html`即可查看演示
